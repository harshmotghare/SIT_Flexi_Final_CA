MERN Stack Restaurant Reservation Application

Project Overview:
This is a full-stack restaurant reservation application built with the MERN stack.
It combines a React frontend, Node.js and Express backend, and MongoDB database for managing reservations seamlessly.

The application allows users to:

#View the restaurant menu and available reservation slots

#Create and manage reservation bookings

#Receive real-time notifications

#Admin views for managing reservations and seating



| Layer       | Technologies                  |
| ----------- | ----------------------------- |
| Frontend    | React, React Router, Axios    |
| Backend     | Node.js, Express.js           |
| Database    | MongoDB, Mongoose             |
| Other Tools | VS Code, Postman, Git, GitHub |
